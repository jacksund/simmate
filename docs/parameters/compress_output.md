
This parameter determines whether to compress the `directory` to a zip file at the end of the run. After compression, it will also delete the directory. The default is False.

=== "yaml"
    ``` yaml
    compress_output: true
    ```
=== "toml"
    ``` toml
    compress_output = true
    ```
=== "python"
    ``` python
    compress_output = True
    ```
