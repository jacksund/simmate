## relax_endpoints
This parameter determines whether the endpoint structures for an NEB diffusion pathway should be relaxed before running the rest of the workflow.

=== "yaml"
    ``` yaml
    relax_endpoints: false
    ```
=== "toml"
    ``` toml
    relax_endpoints: false
    ```
=== "python"
    ``` python
    relax_endpoints: false
    ```
