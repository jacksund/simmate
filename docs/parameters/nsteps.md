## nsteps
This parameter sets the total number of steps for the calculation. For instance, in molecular dynamics workflows, the simulation will stop after this many steps.

=== "yaml"
    ``` yaml
    nsteps: 10000
    ```
=== "toml"
    ``` toml
    nsteps = 10000
    ```
=== "python"
    ``` python
    nsteps = 10000
    ```
