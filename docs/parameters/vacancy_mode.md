## vacancy_mode
For NEB and diffusion workfows, this determines whether vacancy or interstitial diffusion is analyzed. Default of True corresponds to vacancy-based diffusion.

=== "yaml"
    ``` yaml
    vacancy_mode: false
    ```
=== "toml"
    ``` toml
    vacancy_mode = false
    ```
=== "python"
    ``` python
    vacancy_mode = False
    ```
